Here is a brief rundown of the code folder:

  • bin: Contains a compiled DLL
  • scripts: Contains the FSharpChart.fsx file
  • source\fsharpchart: Contains all the source control files.
  • source\generator: Contains some fsx files used to generate the original source code. The Generator.Script.fsx file can be used to create a single FSharpChart.fsx file from all the individual source control files.
  • samples\scripts: Contains some sample script files that demonstrate creating various charts, from the simple to the complex.
  • samples\winforms: Contains a WinForms application that displays a chart and demonstrates how to change properties at runtime.
  • samples\wpf: Contains a WPF application that display a chart.

The table below outlines the types of available charts and the documentation location:

Chart Type Documentation Description
Point Point Chart Uses points to represent data points.
FastPoint FastPoint Chart A variation of the Point chart type that significantly reduces the drawing time of a series that contains a very large number of data points.
Bubble Bubble Chart A variation of the Point chart type, where the data points are replaced by bubbles of different sizes.
Line Line Chart Illustrates trends in data with the passing of time.
Spline Spline Chart A Line chart that plots a fitted curve through each data point in a series.
StepLine Step Line Chart Similar to the Line chart type, but uses vertical and horizontal lines to connect the data points in a series forming a step-like progression.
FastLine FastLine Chart A variation of the Line chart that significantly reduces the drawing time of a series that contains a very large number of data points.
Bar Bar Chart Illustrates comparisons among individual items.
StackedBar Stacked Bar Chart Displays series of the same chart type as stacked bars.
StackedBar100 100% Stacked Bar Chart Displays multiple series of data as stacked bars. The cumulative proportion of each stacked element is always 100% of the Y axis.
Column Column Chart Uses a sequence of columns to compare values across categories.
StackedColumn Stacked Column Chart Used to compare the contribution of each value to a total across categories.
StackedColumn100 100% Stacked Column Chart Displays multiple series of data as stacked columns. The cumulative proportion of each stacked element is always 100% of the Y axis.
Area Area Chart Emphasizes the degree of change over time and shows the relationship of the parts to a whole.
SplineArea Spline Area Chart An Area chart that plots a fitted curve through each data point in a series.
StackedArea Stacked Area Chart An Area chart that stacks two or more data series on top of one another.
StackedArea100 100% Stacked Area Chart Displays multiple series of data as stacked areas. The cumulative proportion of each stacked element is always 100% of the Y axis.
Pie Pie Chart Shows how proportions of data, shown as pie-shaped pieces, contribute to the data as a whole.
Doughnut Doughnut Chart Similar to the Pie chart type, except that it has a hole in the center.
Stock Stock Chart Displays significant stock price points including the open, close, high, and low price points.
Candlestick Candlestick Chart Used to display stock information using high, low, open and close values.
Range Range Chart Displays a range of data by plotting two Y values per data point, with each Y value being drawn as a line chart.
SplineRange Spline Range Chart Displays a range of data by plotting two Y values per data point, with each Y value drawn as a line chart.
RangeBar Range Bar Chart Displays separate events that have beginning and end values.
RangeColumn Range Column Chart Displays a range of data by plotting two Y values per data point.
Radar Radar Chart A circular chart that is used primarily as a data comparison tool.
Polar Polar Chart A circular graph on which data points are displayed using the angle, and the distance from the center point.
ErrorBar Error Bar Chart Consists of lines with markers that are used to display statistical information about the data displayed in a graph.
BoxPlot Box Plot Chart Consists of one or more box symbols that summarize the distribution of the data within one or more data sets.
Renko Renko Chart Displays a series of connecting vertical lines where the thickness and direction of the lines are dependent on the action of the price value.
ThreeLineBreak Three Line Break Chart Displays a series of vertical boxes, or lines, that reflect changes in price values.
Kagi Kagi Chart Displays a series of connecting vertical lines where the thickness and direction of the lines are dependent on the action of the price value.
PointAndFigure Point and Figure Chart Disregards the passage of time and only displays changes in prices.
Funnel Funnel Chart Displays in a funnel shape data that equals 100% when totaled.
Pyramid Pyramid Chart Displays data that, when combined, equals 100%.
